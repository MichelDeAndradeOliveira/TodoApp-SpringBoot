# TodoApp-SpringBoot

Tools Used:
  STS 4
  Postman

Prerequisites
  Spring Tool Suite 4
  Java11

Installation
  Download the repo
  Open CMD in the directory, then write the command " mnv clean install package "
  Or in the STS4 simply run with Boot Dashboard
